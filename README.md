# prettytraceback

Make Python tracebacks pretty everywhere.

Installs the `IPython.core.ultratb` handler to run everywhere.


## Installation

```sh
sudo pip install prettytraceback
sudo python -m prettytraceback install
```

## Removal

```sh
sudo python -m prettytraceback uninstall
sudo pip uninstall prettytraceback
```

## Usage

```sh
python -c 1/0
python prettytraceback/test/test.py
```
